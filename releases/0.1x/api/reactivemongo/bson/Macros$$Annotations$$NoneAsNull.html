NoneAsNull

case class NoneAsNull() extends Annotation with StaticAnnotation with Product with Serializable

Indicates that if an Option property is empty, it will be represented by BSONNull rather than being omitted.

import reactivemongo.bson.Macros.Annotations.NoneAsNull

case class Foo(
  title: String,
  @NoneAsNull description: Option[String])
